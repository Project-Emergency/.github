## 📝 서비스 소개
- 대한민국 응급의료의 중요한 이슈 중 하나인 ‘응급실 뺑뺑이’ 해결을 위한 응급실 요약 AI 서비스입니다
- 사용자가 현재 위치와 불편한 부위를 입력하면 AI가 공공 데이터 포털의 전국 응급실 데이터를 바탕으로 사용자의 증상에 맞는 진료가 가능한 가장 가까운 응급실을 요약하여 추천해주는 것이 주기능입니다
